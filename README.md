# 🚀 Boosting Algorithms in Machine Learning

## 🧠 Overview
This repository contains a comprehensive implementation of Boosting Algorithms in Machine Learning, including:

- AdaBoost  
- XGBoost  
- CatBoost  

The goal is to understand how boosting techniques improve model performance and how each algorithm behaves in both classification and regression problems.

Each notebook includes full pipelines from data preprocessing → training → evaluation → visualization.
